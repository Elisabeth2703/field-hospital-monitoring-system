# Система моніторингу польових госпіталів

Веб-додаток для управління пацієнтами та ресурсами польових госпіталів.

## Команда розробників

- **Єлизавета**: Підсистема керування пацієнтами, класифікація випадків та типів поранень за певний період часу, аналіз часових рядів, MQTT моніторинг, аналітика, база даних пацієнтів.
- **Андрій**: Підсистема керування устаткуванням, аналіз стану медикаментозного поповнення, прогнозування необхідних медикаментів та лабораторного устаткування, база даних устаткування, отримання даних з QR коду лабораторного обладнання.

## Опис проєкту

Система призначена для:
- Реєстрації та моніторингу пацієнтів
- Класифікації поранень та випадків
- Аналізу часових рядів (тиждень, 3 місяці, рік)
- Отримання даних з браслетів через MQTT
- Управління медикаментами та устаткуванням
- Генерації звітів

## Технології

- **Frontend**: React.js, Recharts, Tailwind CSS
- **Backend**: Node.js, Express, MQTT.js
- **Database**: PostgreSQL
